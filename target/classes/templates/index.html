Presentation du projet

Ce systeme apprend a partir du dataset Nursery (UCI Machine Learning Repository), construit un arbre de decision ID3 avec entropie et gain d'information sur des attributs categoriels, puis predit la classe de recommandation pour l'inscription en creche d'une nouvelle candidature.

5 Classes
~10k Entrainement
~2,6k Test
8 Attributs
12 960 Instances totales

Pipeline du projet

  1. Chargement CSV : 8 attributs categoriels + classe cible
  2. Encodage discret : chaque modalite → indice entier par attribut
  3. Arbre ID3 : entropie + gain d'information (Chapitre 5)
  4. Evaluation : exactitude, precision, rappel, F1, matrice de confusion
